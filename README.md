
触发式缓存
========================================
用于触发式的更新缓存的机制, 对缓存进行碎片化处理.


使用方式
----------------------------------------

对应的 `Model` 类需要继承 



安装
----------------------------------------
```
"require" : {
    "gavin-k/yii2-invalid-cache-behavior":"*"
},

```

或者使用命令: 
```
composer require gavin-k/yii2-invalid-cache-behavior
```
